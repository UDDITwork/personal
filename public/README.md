# PATMASTER Public Frontend

Professional, minimalist interface for document intelligence.

## Design Principles

- **Minimalist**: Clean, Apple-like design with no clutter
- **Accessible**: Anyone can use without technical knowledge
- **Powerful**: The tool's power is felt through experience
- **Professional**: No emojis, just clean typography and smooth interactions

## Features

- Drag and drop document upload
- Real-time processing status
- Instant results with metrics
- Image extraction and display
- Complete analysis viewer
- Data export

## Usage

Simply drop a PDF or DOCX file and let the AI extract intelligence.

## Tech Stack

- Pure HTML, CSS, JavaScript
- Inter font family
- Responsive design
- No frameworks needed
